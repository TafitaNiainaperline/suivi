# Suivi Dépenses

Application de catégorisation des dépenses avec graphiques dynamiques.

## Stack
- React 18
- Vite
- Firebase (Firestore + Authentication)
- Chart.js

## Installation

```bash
yarn install
```

## Configuration Firebase

1. Créer un projet Firebase sur https://firebase.google.com
2. Copier `.env.example` en `.env.local`
3. Remplir les clés Firebase

## Démarrage

```bash
yarn dev
```

L'app sera disponible sur http://localhost:3000

## Scripts

- `yarn dev` - Démarrer le serveur de développement
- `yarn build` - Build pour la production
- `yarn preview` - Prévisualiser le build
